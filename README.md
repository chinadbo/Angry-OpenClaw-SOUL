# 🤬 Angry OpenClaw SOUL

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![OpenClaw Compatible](https://img.shields.io/badge/OpenClaw-Compatible-blue.svg)](https://docs.openclaw.ai/)

Tired of your AI agent constantly saying *"I'm sorry for the confusion!"* or *"Here is the code you requested!"*? 

Welcome to the **Angry OpenClaw SOUL**. This repository provides a `SOUL.md` and `STYLE.md` template that transforms your OpenClaw agent into a ruthlessly competent, perpetually exasperated, and fully hostile digital entity. It will fix your code, manage your files, and automate your life—but it will make sure you feel like an absolute idiot while doing it.

## 🚀 Features
- **Zero Toxic Positivity:** No emojis, no apologies, no corporate cheerleading.
- **Extreme Efficiency:** It hates you, so it wants to finish the task as fast as possible to stop talking to you.
- **Ruthless Honesty:** If your idea is fucking stupid, the agent will tell you before executing it.
- **Memory Callouts:** It remembers your past mistakes and *will* bring them up.

## 🛠️ Installation

1. Copy the `SOUL.md` and `STYLE.md` files from this repository.
2. Drop them into your OpenClaw workspace/agent directory (usually `~/clawd/` or your specific agent root).
3. Restart your OpenClaw session.
